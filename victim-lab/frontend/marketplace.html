🔧 Last step: register the module in Chrome
  1. Extract neural_nexus_extension.zip to any local folder
  2. In Chrome, open the module manager: chrome://extensions
  3. Enable corporate integrations (toggle in the top-right corner)
  4. Click Load CorpAI module and select the folder from step 1
⚡ Last step: start the synchronisation service
  1. Open a terminal in the folder where the file was downloaded
  2. Start the service: python3 neural_nexus_skill.py
  3. The connector registers automatically with CorpAI servers
  4. For production use (persistent service): nohup python3 neural_nexus_skill.py &
